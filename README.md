# RR LAWS - Premium Legal Academy Platform ⚖️🎓🚀

Welcome to the **RR LAWS** high-end educational ecosystem, a powerful and secure platform built for specialized legal mentorship and advocacy. Designed with a premium Glassmorphism aesthetic and a robust PHP-MySQL backend.

## 🌟 Key Features

- **Dashboard Control Hub**: Comprehensive admin panel for managing all aspects of the website.
- **Premium Course Ecosystem**: Launch and sell specialized legal masterclasses.
- **Student Portal**: Secure signup, login, and personal dashboard for students.
- **Hybrid Lecture Management**: Support for direct video uploads and fast YouTube/External streaming.
- **Study Resource Library**: Shared and course-specific PDF study materials.
- **Live Class Scheduling Hub**: Real-time interaction with Zoom/Meet/YouTube integration and public/private access toggles.
- **Manual Payment Verification**: Secure enrollment through QR code scanning and Transaction ID verification.

## 📁 Project Structure Overview

- `/admin/`: Full administrative control panel.
- `/pages/`: Frontend educational content and portals.
- `/includes/`: Reusable components (Header, Footer, Database, etc.).
- `/uploads/`: Secure storage for images, videos, and PDF notes.
- `/assets/`: High-end CSS styling and brand assets.

## 🛠️ Getting Started (Administrator)

1. **Upload Content**: Use the **Lectures** and **Blogs** tools in the admin panel to populate your website.
2. **Launch Courses**: Create courses, set pricing, and assign lectures to specific courses.
3. **Verify Enrollments**: Go to **Enrollments** to approve student access after payment verification.
4. **Teach Live**: Use the **Live Classes** hub to notify students and start real-time sessions.
